# Employee Management (Demo)

Simple Spring Boot demo application for managing employees.

## Run

Build and run with Maven:

```bash
mvn clean package
mvn spring-boot:run
```

Or run the packaged jar:

```bash
java -jar target/demo-0.0.1-SNAPSHOT.jar
```

## Notes
- Java/Maven project. See `pom.xml` for dependencies and Java version.
- Database file (H2) may be generated under `data/`.
